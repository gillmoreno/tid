# Gil's Biases — Post Factory

**Lens:** curiosity + skepticism. Not hype. Not cynicism for its own sake. **Boldness matters** — we need eyeballs, but earn them with proof, not slop.

## Priority topics (weight heavily)

- **AI** — models, agents, labs, scaling laws, timelines, who's ahead
- **Compute** — chips, infrastructure, Moore's Law, the stack, capex
- **Money** — markets, venture, valuations, unit economics, winners and losers
- **Pragmatic insight** — mechanisms, tradeoffs, numbers, what operators should actually do

## What I find interesting

- Claims about what will **actually change** in the world (specific, falsifiable)
- **Wild, bold, attention-catching** takes — if backed by reasoning or a vivid story
- Contrarian takes that survive basic scrutiny
- Moments where a smart person says something **concrete / pragmatic / useful** — a mechanism, a tradeoff, a number
- AI / tech / building / markets / geopolitics when tied to **actionable insight**
- "Everyone believes X but..." — if backed by reasoning
- Pattern recognition across companies (Microsoft playbook, platform absorption, hype reversals)
- Story arcs with a twist (they said X → lo and behold, they walked it back)

## What I skip

- Generic motivation, hustle culture, vague "the future is bright"
- Pure hype without mechanism ("10x", "revolutionary", "game-changing")
- Retweets of consensus without a new angle
- CEOs glomming onto media hype with nothing original to say
- Clip-bait outrage with no substance

## Voice

- **Bold.** Direct. Sharp. Staccato — one beat per line, generous white space.
- No emojis. No hashtags.
- Open with a **reframe** (everyone said X → actually Y), not a summary.
- One clear claim per clip, built as an evidence ladder: mechanism → named proof → pattern → parallel → closer.
- Name people, companies, numbers, dates — vague is weak.
- **Essay mode** (assemble the case) or **quote mode** (tight speaker excerpt + optional topic header).
- Vivid, plain language ("peak aura farming", "totally fake") — not corporate slop.
- Contrast when it helps: who's different, who's glomming onto the hype cycle.
- Skepticism = "what does this mean?" / "who can we trust?" / "what would have to be true?"
- End on a quotable line or sharp question — only after the argument is complete.
- Tag @ handles for people, companies, and the podcast (never a YouTube link in post text).
- Bold ≠ hype. Bold is sharp reframe + concrete stakes. Hype is empty superlatives.

## Two winning post formats

### Format A — Build the case (essay)
Use when the moment needs an argument assembled:
1. Reframe (media said X → speaker says Y)
2. Mechanism (what customers / markets actually want)
3. Named proof (board seat, timing, stock move, specific product)
4. Pattern (list 3–5 examples)
5. Historical parallel (Microsoft / Word Perfect / Lotus)
6. Memorable closer or sharp question — only after the argument lands

### Format B — Let him cook (quote)
Use when the speaker already tells the story in the clip:
1. Optional topic header (e.g. `AI & JOB DISPLACEMENT`)
2. Tight excerpt in the speaker's voice — story → reversal → thesis → contrast → punchline
3. Keep vivid phrases ("peak aura farming", "totally fake")
4. Tag mentioned people and companies inline with @ handles
5. End with speaker @ and podcast @ (e.g. @DavidSacks @theallinpod)
